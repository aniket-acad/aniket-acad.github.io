Hash Table

Grading Policy

  • Deduce 0.5 marks for every mistake.
  • The solution must be presented in a tabular format.

Using the following hash function, demonstrate the state of hash table after all the values listed below are inserted

Hash function: Convert the input value to a binary format of 5 digits. Select the binary digits at index 0, 2 and 4. Form a numerical value by putting the selected binary digits together. Use this numerical value as key for the hash table.

For example, value 25 is 11001 in binary. Binary digits at index 0, 2 and 4 = 1 0 1, which is 5 in numeric value. Thus hash(25) = 5, i.e. 5 is the key.
